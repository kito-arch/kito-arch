### What I do
Design and develop scalable web applications and backend systems using JavaScript, Node.js, React, and GraphQL. Strong expertise in cloud platforms (AWS, Azure), API architecture, automated testing, and system optimization. Experience working on data visualization platforms (Apache Superset), implementing secure authentication using OIDC, developing Model Context Protocols (MCPs) for GraphQL services, and building Retrieval-Augmented Generation (RAG) pipelines. Proven ability to modernize legacy systems, improve performance, and deliver production-ready solutions.
